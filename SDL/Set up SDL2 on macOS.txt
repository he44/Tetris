Set up SDL2 on macOS

1. Go to: https://www.libsdl.org/ 
   download the package

2. Open the package

3. Open a finder window, press Cmd + Shift + g, type /Library/Frameworks 

4. Copy the SDL file into /Frameworks

5. Get the sample code from 
http://lazyfoo.net/tutorials/SDL/01_hello_SDL/index2.php

6. Compile:

gcc main.c -F/Library/Frameworks/SDL2.framework/Headers -framework SDL2

7. run it
